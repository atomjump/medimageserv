Quick Start Information

MedImage Server offers a choice of three ways to connect to your computer. You can add new phones by pressing one of the option buttons above, and if you come back later, you can add more phones with the same option.

  • WiFi. Connect through a wifi network attached to this server. This is very fast and free to use but works only in range of the wifi network.
  • Use our servers. Usage charges apply after an initial 5GB of free bandwidth (note: this is the total free bandwidth, not a monthly figure), but are very reasonable.
  • Use your own web server. See the 'Proxy server' setup for instructions.
